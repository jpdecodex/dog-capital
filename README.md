# Dog Capital

Public marketing site for Dog Capital — systematic, rules-based investment
programs for Argentine investors. Live at
[jpdecodex.github.io/dog-capital](https://jpdecodex.github.io/dog-capital).

## Stack

Static site built with [Quarto](https://quarto.org) and deployed to GitHub
Pages. No JS framework.

## Structure

```
index.qmd              Homepage — single scrolling page (About Us, Programs,
                        Research, Contact, Legal), all navigation is anchor
                        jumps within this one page.
programs/               Deep program pages — one per investable program,
                        linked from the homepage's Programs section.
brand/                  Brand reference material (manual, logo source files) —
                        not part of the rendered site.
styles.css               Site-wide styles.
_quarto.yml              Project/site config.
docs/                    Render output — this is what GitHub Pages serves.
                        Generated by `quarto render`, do not hand-edit.
```

## Local development

```
quarto preview
```

Renders and serves the site locally with live reload. To produce the
`docs/` output that GitHub Pages serves:

```
quarto render
```

Commit the `docs/` output alongside any source change — it's checked in,
not built by CI.

## Content and disclosure

This is a public-facing site. Do not add real account balances, entry/exit
levels, prop-firm or funded-account mechanics, or non-public partner names.
Venue names for the systematic programs (e.g. IBKR, IOL) are fine to state —
standard track-record transparency. See `CLAUDE.md` for the fuller working
history and open decisions behind the current copy and design.
