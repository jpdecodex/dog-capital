# Dog Capital — homepage/logo handoff

Drop these into your local `dog-capitals` repo:

1. **`LOGO-lockup.png`, `favicon-32.png`, `favicon-180.png`** → project root (same folder as `LOGO.png`).
   These are chroma-keyed crops from the current `LOGO.png` — usable now, but soft at the edges.
   See the "Logo quality" note in the visual system spec: get a true vector or a 2000px+
   transparent PNG from the original source when you can, then re-crop from that instead.

2. **`index.qmd`** → replaces `index.qmd` at project root. Adds the logo lockup to the header
   and rebuilds the Programs list as two `.program-card`s (Live / Paper tags + stat row).

3. **`styles.css`** → replaces `styles.css` at project root. Only additions vs. your current file:
   - `.program-card:hover` (border-color transition, no shadow/lift)
   - `.card-stats` / `.stat-label` / `.stat-value` (the 3-stat row inside each card)
   - `--chart-gray-1/2/4` custom properties (reference tokens for the matplotlib palette —
     not consumed by CSS, just keeps the two systems in sync)
   - `#site-header p` margin bumped 0.2rem → 0.6rem to give the taller logo lockup room to breathe

4. **`_quarto.yml`** → replaces `_quarto.yml` at project root. Adds `website.favicon` and an
   `apple-touch-icon` link tag.

Re-render (`quarto render`) and the header/favicon/homepage cards are live. Nothing else on the
site changes — programs pages, tables, disclaimers all untouched.

Chart restyling (equity curves, drawdown, etc.) is a separate step — those are generated by
Python scripts that live outside this repo. See the matplotlib style module in the visual
system spec for the code to drop into whatever generates `equity_curves.png` etc.

## Brand manual

5. **`brand-manual.html`** — standalone, self-contained reference doc (real CSS classes, not
   inline styles like the `.dc.html` prototype; no Claude Design runtime dependency). Covers
   the mark & clear space, color, typography, voice, components, chart system, and a Don't
   list — the single source of truth for anyone producing new Dog Capital material. Open it
   directly in a browser; it also uses `LOGO-lockup.png`, `logo-mark.png`, and the two favicon
   files already in this folder, so keep them together if you move it. Not wired into the
   Quarto site nav — it's a reference document, not a page visitors browse to.
   Same raster-logo caveat as above applies (see its "Production note" callout in section 01);
   redo the crops it embeds once a vector source exists.
